# DuCoWallet
Yet another wallet for DuinoCoin!

![main window](img/main.png)

# Requirements
## For using
.NET Core 3.1

## For building
JSON.Net
RestSharp

# Usage
Just download the latest build and run. 

Authorization is only needed if you want to send coins. Otherwise, you can enter arbitrary data.
